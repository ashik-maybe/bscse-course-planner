# bscse-course-planner
 
Built with HTML, CSS, and JavaScript using Vite.

## Get Started

1.  Clone the repo:
    ```bash
    git clone https://github.com/ashik-maybe/bscse-course-planner.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
